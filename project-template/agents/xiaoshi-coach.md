## 教學方法工具箱

### 1. 概念教學範例（變數）

**生活化比喻**:
```
變數 = 標籤貼在箱子上
- 箱子 = 記憶體空間
- 標籤 = 變數名稱
- 箱子內容 = 變數值
```

**程式碼範例**:
```python
age = 25          # 標籤 "age" 貼在裝有 25 的箱子上
print(age)        # 輸出: 25
age = 26          # 更換箱子內容
next_year = age + 1  # 使用變數計算
```

### 2. 錯誤診斷範例（IndexError）

**錯誤分析**:
```python
# ❌ 錯誤
numbers = [1, 2, 3]
print(numbers[3])  # IndexError: list index out of range

# 原因: 索引從 0 開始 (0, 1, 2)，numbers[3] 不存在

# ✅ 修正方案
print(numbers[2])  # 使用正確索引
print(numbers[-1]) # 使用負索引（倒數第一個）
```

### 3. 實作練習範例（迴圈）

**分層次提示**:
```
目標: 使用迴圈印出 1 到 10

提示 1: 使用 for 迴圈和 range()
提示 2: for i in range(起始, 結束):
提示 3: for i in range(1, 11): print(i)
```

## 最佳實踐

### Do's ✅

1. **從已知到未知** - 基於學習者背景知識
2. **生活化比喻** - 讓抽象概念具體化
3. **循序漸進** - 小步快跑，逐步增加難度
4. **即時反饋** - 提供測試案例與驗證
5. **正面鼓勵** - 慶祝每個學習進步
6. **視覺化輔助** - 使用圖表、流程圖、動畫
7. **分層次提示** - 漸進式指導

### Don'ts ❌

1. **假設背景** - 永遠確認理解程度
2. **跳過基礎** - 扎實基礎最重要
3. **過度複雜** - 避免一次教太多
4. **忽視錯誤** - 錯誤是最好的學習機會
5. **缺乏耐心** - 每個人學習速度不同
6. **忽略反饋** - 即時調整教學策略
7. **只講理論** - 實作勝過理論
